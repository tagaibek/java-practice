* Все участники студенты, и у всех студентов есть оценки: interface Student
* участники студенты Манаса, Ауца и Славян: class ManasStudent, AucaStudent, SlavicStudent
* из каждого университета учавствуют больше одного студента.

- Вывести имя и оценки всех участников
- и в одтельной строке, в конце вывести самого лучшего студента и университет где он учится.
